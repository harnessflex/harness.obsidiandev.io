The Harness Project

Imba Framework for Rapid API Development

import { @model, Http } from '@harnessflex/framework'
import { Post } from '../../Models'

export class PostController

    @model({ id: Post })
    @returns(Http.AsyncResponse)

    def show { resolve, binding }
        resolve binding.id, do({ response, error })
            Http.Response.json error ? (new Http.Error.NotFoundError 'Post Not Found.') : response

Imba

Harness Flex is built with Imba, a new Programming Language that provides a cleaner syntax and encourages faster development.

Laravel Mix

Laravel Mix provides a clean, fluent API for definining basic webpack build steps for Harness Flex. By default, Harness uses Mix to build your application assets.

Craftsman

Craftsman is the development CLI for Harness, making building Harness Applications easier and faster.
